Project Description

The Shelter Records System is a comprehensive web application developed for Greater Chennai Corporation to efficiently manage and maintain records of inmates and staff across various shelter facilities. The system provides a centralized platform for record keeping, staff management, and facility oversight.

Built with Django framework for robust backend functionality, the application features a responsive frontend using HTML, CSS, and JavaScript. The system integrates advanced face recognition technology for secure staff identification and attendance tracking, ensuring accurate record maintenance.

The Story

Greater Chennai Corporation needed a modern, efficient system to manage their shelter facilities and maintain comprehensive records of both inmates and staff. The existing manual processes were time-consuming and prone to errors. The challenge was to create a secure, user-friendly platform that could handle large volumes of data while providing real-time access to critical information.

Our Approach

We developed a Django-based web application with PostgreSQL for primary data storage and Milvus DB for efficient face recognition vector storage. The system features role-based access control, real-time data synchronization, and an intuitive interface for easy navigation. Face recognition technology was integrated using advanced algorithms to ensure accurate staff identification and attendance tracking.

Project Description

The Face Attendance App is a sophisticated mobile application designed to streamline staff attendance tracking for Greater Chennai Corporation's shelter facilities. The app leverages cutting-edge facial recognition technology to provide secure, contactless attendance management.

Built with Django REST Framework for robust API functionality, the application integrates DeepFace library with ArcFace algorithm for highly accurate face detection and recognition. The system ensures real-time attendance tracking while maintaining data security and privacy standards.

The Story

Greater Chennai Corporation required an automated attendance system for their shelter staff that could eliminate manual processes and reduce administrative overhead. The challenge was to create a reliable, secure, and user-friendly mobile application that could accurately identify staff members and track their attendance in real-time.

Our Approach

We developed a mobile application with Django REST Framework backend API that integrates DeepFace library for facial recognition. The ArcFace algorithm was specifically chosen for its superior accuracy in face detection and recognition. The app features secure authentication, real-time data synchronization, and comprehensive attendance reporting capabilities.

Project Description

The Custom Song Platform is an innovative web application that revolutionizes the way people create personalized music. Users can order custom songs based on their life stories, relationships, and special moments. The platform provides a complete ecosystem for song creation, customer management, and delivery.

Built with Django for robust backend functionality and HTML/CSS/JavaScript for a modern, responsive frontend, the platform integrates MySQL database for efficient data management. The system features Django REST Framework APIs for seamless data handling, WhatsApp API for customer communication, and Razorpay integration for secure payment processing.

The Story

The idea was to create a platform where people could turn their life stories, memories, and emotions into personalized songs. The challenge was to build a comprehensive system that could handle the entire process from order placement to song delivery, while providing excellent customer service and maintaining detailed records of all transactions and customer interactions.

Our Approach

We developed a full-stack web application using Django framework with MySQL database for reliable data storage. The platform includes a customer dashboard for order management, a relationship manager portal for staff to maintain customer details and track collections, and integrated payment processing through Razorpay. WhatsApp API integration enables seamless communication with customers for sharing customized songs and updates.
